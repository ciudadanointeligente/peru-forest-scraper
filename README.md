Peru forest scraper
===================

Gathers forest information from Peru and publishes it to the billit-api project

Run
----

You need the datasets in CSV format, using the '|' delimiter. To launcher the script you need Ruby 1.9.

    $ ruby scraper.rb

# Dependencies

gem install rest-client
